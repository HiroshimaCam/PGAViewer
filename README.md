# インストール方法
* 解凍したファイルをchrome://extensionsから拡張機能として追加して下さい。
# 機能
* 強震モニタの画像を解析して、揺れを検出したときに音や通知で地震が発生したことをお知らせします。
* 揺れ検出に特化したアプリなので、本家強震モニタのように緊急地震速報や到達予想円などの情報は表示されません。
# 設定
* 設定できるのは音と通知のみです。誤検出も稀にあるため、弱い反応時の通知や音声はデフォルトでオフになっています。
* 設定は保存されません。
# 揺れ検出システムの話
* 強震モニタの地表加速度・地中加速度の画像を1秒ごとに取得・解析し、
* ・地表で黄色・赤の観測点がある
* ・その周辺の観測点で緑、黄色、赤の観測点が8割を超えている
* ・地中で反応している観測点が3点以上ある
* 時に地震が発生したと判断します。
* 首都圏・桜島周辺はほかの地域に比べて反応している観測点が多いため、フィルターをかけています。
* 強震モニタ側の規約で、震度換算ができないため、取得したデータはあくまでも色として扱うようにしています。
# 問題点
* 観測点の故障などで過剰に反応している観測点があるときに、その周辺で地震が発生すると反応を過大に評価してしまいます。
* そこまで大きな地震でなくても加速度では赤反応が出ることがあるので、震度２～３の地震でも「強い揺れを検出しました」とでることがあります。
