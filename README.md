# インストール方法
* 解凍したファイルをchrome://extensionsから拡張機能として追加して下さい。
# 機能
* 強震モニタの画像を解析して、揺れを検出したときに音や通知で地震が発生したことをお知らせします。
* 揺れ検出に特化したアプリなので、本家強震モニタのように緊急地震速報や到達予想円などの情報は表示されません。
# 設定
* 設定できるのは音と通知のみです。誤検出も稀にあるため、弱い反応時の通知や音声はデフォルトでオフになっています。
* 設定は保存されません。
# 揺れ検出システムの話
* 強震モニタの地表加速度・地中加速度の画像を1秒ごとに取得・解析し、地表のデータで黄色・赤色の観測点を発見したら周辺の観測点を走査し、青を除いた緑、黄色、赤の観測点が8割を超えて、なおかつ地中のデータで反応している観測点が3点以上の時に地震が発生したと判断します。
